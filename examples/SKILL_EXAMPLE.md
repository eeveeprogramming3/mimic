# Open Browser and Search

## Description
Opens a web browser, navigates to a search engine, and performs a search query.

## Prerequisites
- Web browser installed (Firefox, Chrome, or similar)
- Desktop environment with GUI
- Internet connection

## Steps

### 1. Open the browser
- Click on the browser icon in the taskbar or application menu
- Wait for the browser window to fully load

### 2. Navigate to search engine
- Click on the URL/address bar at the top of the browser
- Type `google.com` or `duckduckgo.com`
- Press Enter to navigate

### 3. Perform search
- Wait for search page to load
- Click on the search input field
- Type the search query: `{query}`
- Press Enter to search

### 4. Verify results
- Wait for search results to load
- Confirm results are displayed

## Input Variables
- `query`: The search term to look up (e.g., "weather today", "python tutorial")

## Expected Output
- Browser window open with search results displayed
- Results relevant to the query shown on page

## Error Handling
- If browser doesn't open: Check if browser is installed
- If page doesn't load: Check internet connection
- If search fails: Try refreshing the page

## Notes
- This skill generalizes to any search query
- Works with most major browsers
- Search engine can be changed by modifying step 2
